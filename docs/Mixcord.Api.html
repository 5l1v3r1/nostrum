    error_map() 
    error_map :: {:error, Map.t}

    Represents a failed response from the API. This occurs when hackney or HTTPoison fail, or when the API doesn’t respond with 200 or 204.

    The map is structured as follows:

    • status_code

      • nil if HTTPoison or Hackney throws an error.
      • Status code of response otherwise.

    • message

      • Error message of response.
